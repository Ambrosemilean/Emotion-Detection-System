# Motion Detection with Emotion Recognition System

A Python-based motion detection system that combines computer vision for motion tracking with IBM Watson NLP for emotion recognition from text input.

## Features
- Real-time motion detection using webcam
- Emotion analysis of text using IBM Watson NLP
- Modular architecture for easy integration
- Comprehensive test suite
